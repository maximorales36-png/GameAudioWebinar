# Game Audio — Recursos & Herramientas

Página web creada para el webinar de Game Audio.

Incluye una lista de recursos y herramientas gratuitas para trabajar en audio para videojuegos, y un generador de nombres de assets de sonido (Mx, Sfx, Amb).
